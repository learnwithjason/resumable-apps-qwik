Qwik lets us build "resumable apps" by requiring nearly zero JavaScript for pages to become interactive + fine-grained lazy loading to only load what's used. Miško Hevery will teach us how to get started.

## Development

### Install

```
npm install
```

### Development Build

```
npm run dev
```

### Production Build

```
npm run build
```
